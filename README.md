Reactor Pattern in C++
===

### Compile && Install
```sh
$ cmake -S . -B build
$ cd build
$ make && make install
```

### Get Started
```sh
$ ./examples/echo/echo_server # server
```
```sh
$ telnet 127.0.0.1 8317 # client
```
